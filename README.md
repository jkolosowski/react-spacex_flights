# SpaceX Flights
App feature all SpaceX flights with CRUD operations. Included advanced sorting and filtering. Also contain information about rockets.

### Used technologies
- React
- React Router
- Redux
- Redux middleware
- Normalizr
- Bulma.io
- Express
- MongoDB
- Docker

Based on API: https://github.com/r-spacex/SpaceX-API
